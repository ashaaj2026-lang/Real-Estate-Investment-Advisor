🏘️Real Estate Investment Advisor: Predicting Property Profitability & Future Value

## Project Overview

This project is a Machine Learning application developed to help potential investors make informed real estate investment decisions.

The application performs two major tasks:

- Predicts whether a property is a **Good Investment** (Classification).
- Predicts the **estimated future property value after 5 years** (Regression).

The project also includes a Streamlit web application for interactive predictions.

---

## Problem Statement

Real estate investors often find it difficult to evaluate whether a property is worth investing in and estimate its future value.

This project uses Machine Learning techniques to analyze property details and provide investment insights based on historical housing data.
